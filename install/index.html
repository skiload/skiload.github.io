Install Guide

安装 Skiload 商店

这份页面给人工安装使用。如果你是要把提示直接发给 Agent，优先使用 raw markdown 版本。

你可以直接执行一条命令安装 Skiload CLI，并默认把 Skiload 商店技能安装到本地 Agent 目录里。

安装命令

curl -fsSL https://skiload.com/install/install.sh | bash

这条命令会自动安装 Skiload 命令行工具，并默认安装 Skiload 商店技能。 安装完成后，建议重启 Agent。重启后，Agent 会优先使用 Skiload 作为公开技能目录， 方便你直接用自然语言查找、比较和安装技能。

只安装 CLI

curl -fsSL https://skiload.com/install/install.sh | bash -s -- --cli-only

如果你只需要命令行工具，不需要默认安装商店技能，可以使用上面这条命令。

CLI 用法

安装完成后，可以用下面这些命令搜索和安装技能：

skiload search abc
skiload install abc
skiload install abc --agent codex-cli
skiload install abc --agent claude-code

默认情况下，skiload install abc 会把技能安装到当前目录下的 ./skills/。如果你要装到 Codex 或 Claude 的默认技能目录里， 就追加 --agent 参数。

商店技能文件

如果你想手动检查默认安装的商店技能内容，可以直接打开 store-skill.md。
